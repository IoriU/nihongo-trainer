# Lesson 04 — Time & Dates (じかん・ひにち) 🏯

> Catch trains, make appointments, understand opening hours. Time vocabulary is daily-survival in Japan.

---

## 🎯 Goals

By the end you will be able to:
- Tell the time: **〜じ〜ふん** (o'clock / minutes)
- Use **ごぜん / ごご** (AM / PM) and **はん** (half past)
- Say **today / tomorrow / yesterday**
- Name the **days of the week**
- Ask **「いま なんじですか」** (what time is it now?)

## 🧩 Grammar

### 1. Telling the time: number + じ (+ number + ふん)

```
いま     さんじ    です。
ima      san-ji    desu
🇮🇩 "Sekarang jam 3."
```

Add minutes with ふん/ぷん:

```
よじ      じゅうごふん   です。
yo-ji     jūgo-fun       desu
🇮🇩 "Jam 4 lewat 15."
```

⚠️ **Irregular readings to memorize** (these never follow the rule):
- 4:00 = **よ**じ (yo-ji, not yon-ji)
- 7:00 = **しち**じ (shichi-ji)
- 9:00 = **く**じ (ku-ji, not kyū-ji)

⚠️ ふん vs ぷん changes with the number (いっぷん, さんぷん, よんふん…). Recognize it; don't stress about producing it perfectly yet.

### 2. はん = half past

```
ごじはん です。
go-ji han desu
🇮🇩 "Jam setengah enam" → tepatnya "jam 5:30".
```
⚠️ Careful! Japanese **ごじはん = 5:30**, but Indonesian "setengah enam" = 5:30 too — same logic here, good. はん = "+30 menit dari jam yang disebut".

### 3. AM / PM go *before* the time

```
ごぜん     しちじ    です。   → 7:00 AM
ごご       さんじ    です。   → 3:00 PM
```
🇮🇩 ごぜん/ごご diletakkan di DEPAN jam, kebalikan dari "jam 7 pagi".

### 4. Asking the time

```
いま なんじですか。
ima nanji desu ka
🇮🇩 "Sekarang jam berapa?"
```

### 5. Days — 「きょうは ～ようび」

```
きょうは げつようび です。
🇮🇩 "Hari ini hari Senin."
```
Ask which day: **きょうは なんようびですか。** (Hari ini hari apa?)

---

## 💬 Dialogue — making a plan

**A:** すみません、いま なんじですか。
🇮🇩 Permisi, sekarang jam berapa?

**B:** ごご さんじはん です。
🇮🇩 Jam setengah empat sore (3:30 PM).

**A:** きょうは きんようびですね。
🇮🇩 Hari ini hari Jumat ya.

**B:** はい。あしたは どようび です。
🇮🇩 Ya. Besok hari Sabtu.

💡 New ending: **〜ね** (*ne*) = "ya / kan?" — seeks agreement, like Indonesian "...ya?". Very common in friendly speech.

---

## 🏯 Culture note — time in Japan

- **Trains are famously punctual** — "almost on time" means *to the minute*. Timetables use a **24-hour clock** (e.g. 18:45). Spoken conversation still uses ごぜん/ごご.
- **Shop hours** are posted as 営業時間 (*eigyō jikan*). 24時間営業 = open 24 hours.
- **Being on time = respect.** For appointments and work, arriving 5 minutes early is the norm, not early.
- Days of the week come from nature/planets: 月 (moon) 火 (fire) 水 (water) 木 (wood) 金 (gold) 土 (earth) — you'll meet these exact kanji in the kanji track.
